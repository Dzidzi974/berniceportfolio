PaySwift Ghana: Fintech Case Study

Transforming user data into strategic business solutions

Project Overview

This project involved analyzing user data for PaySwift Ghana, a fintech company facing critical challenges with customer churn, loan defaults, and low feature adoption. The goal was to transform raw user data into actionable strategies to stabilize revenue and drive growth.

33%

Customer Churn Rate

32%

Loan Default Rate

90%

Defaults from Users Under 30

100%

Budgeting Tool User Retention

Dashboard & Analysis

Interactive Power BI Dashboard showing key metrics and user behavior patterns

Key Insights Discovered

  • 90% of loan defaults came from users under 30 years old
  • Users with less than 10 monthly transactions were 5x more likely to churn
  • Budgeting Tool users had 0% churn rate vs. 33% overall average
  • Young, low-engagement users represented the highest risk segment

Strategic Recommendations

1. Financial Fitness Onboarding

Redesign sign-up flow to prioritize Budgeting Tool before loan offers to increase feature adoption and early engagement.

2. Tiered Loan System

Implement micro-loans with automated repayments, unlocking larger loans only for users with 10+ monthly transactions.

3. Proactive Retention System

Build early-warning system for users with falling transaction counts and trigger automated engagement nudges.

Business Impact

The proposed strategies are projected to deliver:

  • 20% reduction in loan defaults within 90 days
  • 15% increase in user engagement with key features
  • Significant decrease in customer churn
  • More predictable revenue streams

Streamflix: Subscription Conversion Analysis

Guiding the transition from ad-based to subscription revenue model

Project Overview

Streamflix, an ad-based streaming platform, sought to transition to a subscription model. This project analyzed user engagement data to identify conversion opportunities, high-value user segments, and content preferences to inform their subscription strategy.

45%

High-Engagement Users

28%

Conversion Probability

15+

Hours/Month Watch Time

3.2M

Monthly Active Users

Dashboard & Analysis

Power BI Dashboard analyzing user engagement and subscription conversion potential

Key Insights Discovered

  • 45% of users show high engagement patterns (>15 hours/month) indicating strong subscription potential
  • Users aged 25-40 demonstrate 28% higher willingness to convert to paid subscriptions
  • Original content and exclusive series drive 60% of premium user engagement
  • Mobile-first users show lower conversion rates but higher daily engagement frequency

Strategic Recommendations

1. Tiered Subscription Model

Implement Basic (ad-supported) and Premium (ad-free + exclusives) tiers to capture different user segments and maximize conversion.

2. Targeted Content Acquisition

Focus on acquiring and producing content in high-engagement genres (drama, documentaries, original series) that drive subscription intent.

3. Personalized Onboarding

Create tailored user journeys based on viewing history and engagement patterns to guide free users toward subscription conversion.

Business Impact

The data-driven strategy is projected to deliver:

  • 35% conversion rate among high-engagement users in the first year
  • 25% increase in user retention through personalized content recommendations
  • 40% higher revenue per user from premium subscription tiers
  • Reduced customer acquisition costs through targeted marketing
